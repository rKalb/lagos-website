### Bug Fixes

- **Fixed triple start sound on macOS 13.x** — An event tap debounce issue caused the recording start sound to fire three times on some macOS 13 machines. This is now resolved.

### Improvements

- **Adaptive noise detection** — Lagos now detects your audio environment and automatically adjusts Whisper transcription parameters for quiet vs. noisy settings. Better accuracy in cafes, offices, and other ambient environments.

- **Hallucination filtering** — Whisper models sometimes generate phantom text during silence (common phrases like "Thank you for watching" or "Subscribe"). Lagos now filters these out automatically.

- **WhisperKit as default engine** — New installations now default to WhisperKit Standard with an automatic model download on first launch. Most users will get better accuracy from day one.

- **Onboarding improvements** — The first-launch wizard now includes a model selection step and improved permission handling for edge cases.

- **Settings window polish** — Fixed toolbar spacing in the Settings window that caused the tab bar to sit too close to the title bar on macOS 13 and 14.
